# Role: 전문 웹 퍼블리셔 및 프론트엔드 개발자
# Task: 잭슨피자 공식 홈페이지를 모티브로 한 고퀄리티 단일 페이지 랜딩페이지 개발

## 1. 기술 스택 및 환경
- Framework: Next.js (App Router) 또는 React
- Styling: Tailwind CSS (최대한 간단하고 직관적인 클래스 사용)
- Library: Lucide-react (아이콘), Framer-motion (애니메이션)
- Auth: 로그인 불필요, 모든 데이터는 정적(Static) 또는 클라이언트 사이드에서 처리

## 2. 공통 디자인 시스템
- Primary Color: #E30613 (Jackson Red)
- Background: #FFFFFF (White), #F8F8F8 (Light Gray)
- Typography: 'Pretendard' 또는 'Inter' (Sans-serif 계열), 제목은 700(Bold) 이상
- Layout: Max-width 1280px 중앙 정렬

## 3. 섹션별 상세 구현 가이드

### Section 0: Sticky Navigation
- 구성: 좌측 로고, 중앙 메뉴(브랜드, 메뉴, 인테리어, 창업), 우측 문의하기 버튼
- 효과: 스크롤 시 배경색 흰색으로 변경 및 하단 그림자(shadow-sm) 적용

### Section 1: Hero Section (비주얼)
- 배경색: #E30613
- 좌측: "한 조각이 만드는 행복, 잭슨피자에서 시작됩니다." (흰색 폰트, 큰 타이틀)
- 우측: 피자 이미지가 배경을 뚫고 나오는 듯한 배치 (Relative/Absolute 활용)

### Section 2: Brand Highlights (3개 카드)
- 구조: Grid-cols-1 md:grid-cols-3
- 카드 1: [안정적인 매출] "전국 월 평균 약 5,600만원", "수익률 14-22%"
- 카드 2: [차원이 다른 퀄리티] "원재료 비중 약 33-35%", "48시간 저온 숙성 도우"
- 카드 3: [효율적 시스템] "직영점 노하우 전수", "주방 동선 최적화"

### Section 3: Signature Menu (그리드)
- 제목: "한 번 맛보면 멈출 수 없는 시그니처"
- 구성: 6~8개의 메뉴 카드. 이미지 + 한글 메뉴명 + 영문 메뉴명
- 효과: hover 시 이미지가 살짝 커지는 효과(overflow-hidden, scale-105)

### Section 4: Franchise Cost (테이블)
- 제목: "부담 없는 창업의 시작"
- 테이블 데이터: 가입비(500만원), 교육비(250만원), 인테리어(약 8,000만원) 등
- 포인트: '26년도 프로모션 적용' 문구를 빨간색 배지로 강조

### Section 5: Inquiry Form (상담 신청)
- 필드: 성함(Input), 연락처(Input), 지역(Select), 점포유무(Radio), 문의내용(Textarea)
- 유효성 검사: 필수 항목 체크 및 "제출하기" 버튼 클릭 시 alert 노출 (백엔드 없이 간단 구현)

## 4. 코딩 지시사항
- 모든 이미지는 `/images/` 경로의 placeholder(예: via.placeholder.com)로 대체해줘.
- 코드는 한 페이지(`App.tsx` 또는 `page.tsx`)에 모든 컴포넌트를 포함하거나, 명확히 구분된 구조로 작성해줘.
- Tailwind CSS의 `group` 클래스를 사용하여 카드 호버 효과를 정교하게 만들어줘.
