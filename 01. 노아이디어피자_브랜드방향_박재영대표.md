# 노아이디어피자가 가야할 방향

> **핵심 키워드**: 코리안 피자 (Korean Pizza)
> **모티브 브랜드**: 잭슨피자 (Jackson Pizza)

---

## 1. 재무 목표 (가맹점 기준)

| 항목 | 목표 비율 |
|------|----------|
| **원가율** | 28~32% |
| **인건비** | 20% 이하 |
| **임대료** | 10~15% |
| **영업이익** | 15~20% |

---

## 2. 브랜드 컨셉: **코리안 피자**

### 왜 "코리안 피자"인가?

- **도우**: 아메리칸 스타일 (뉴욕피자 = 조각피자 이미지지만, 판피자 형태)
- **잭슨피자**: 이미 판피자 시장에서 포지셔닝 확립
- **노아이디어피자 시그니처 메뉴**: 이미 **코리안 스타일**에 맞춰 구성됨
  - 꽈리불고기, 제육피자 등 한국적 토핑 활용

> **결론**: 컨셉을 **"코리안 피자"**로 확고히 가져간다.

---

## 3. 메뉴 리뉴얼 방향

### 3.1 피자 메뉴 변경

| 변경 유형 | 기존 메뉴 | 신규/대체 메뉴 |
|----------|----------|---------------|
| **OUT** | 불닭고구마 | - |
| **OUT** | 핫치킨밤 | - |
| **IN** | - | 제육피자 |
| **IN** | - | 더블치즈베이컨 |

### 3.2 도우/토핑 변경

| 항목 | 변경 내용 |
|------|----------|
| **크리스피넛** | OUT → 오리지널 도우로 단순화 |
| **통베이컨** | 오뗄 제품으로 대체 |

### 3.3 사이드 메뉴 변경

| 기존 | 변경 |
|------|------|
| 콘 샐러드 | 펌킨 그릭요거트로 대체 |
| 머쉬룸크림파스타 | **불고기크림파스타**로 네이밍 변경 (베이컨 X → 불고기 토핑) |

---

## 4. 운영 최적화 전략

### 4.1 메뉴 단순화 원칙

- **수요 적은 메뉴** → 과감히 제거
- **점바점(점포별) 맛 차이 발생 메뉴** → 제거
- **재료 관리 어려운 토핑** → 제거
- **동일 재료, 토핑만 변경** → 맛의 변화 + 원가율 DOWN

### 4.2 주방 오퍼레이션

| 목표 | 상세 |
|------|------|
| **지속적 단순화** | 레시피, 조리 과정 간소화 |
| **맛의 일관성** | 어느 매장에서도 동일한 맛 |
| **품질 유지 용이** | 가맹점 직원 숙련도 낮아도 수월하게 운영 가능 |

---

## 5. 노아이디어피자의 차별화 강점

### 5.1 매출 구조

> **홀 + 포장 비율 최대화** → 배달 매장보다 **수익률이 좋은 브랜드**로 인식되게

### 5.2 인테리어 전략

| 요소 | 방향 |
|------|------|
| **스타일** | 힙(Hip)하지만 복제 가능하게 |
| **사인 포인트** | 고정화 (브랜드 통일성) |
