# 폴리쉬(Poolish) 도우 레퍼런스

> 작성일: 2026-01-30 | 콘텐츠 활용: 블로그, 숏폼, 랜딩페이지 등

---

## 폴리쉬(Poolish)란?

**프랑스식 사전발효(Pre-ferment) 기법**으로, 1840년대 폴란드에서 개발되어 비엔나 제빵사들을 통해 프랑스로 전파된 정통 제빵 기술이다.

밀가루와 물을 **1:1 비율(100% 수화율)**로 섞고 소량의 효모를 넣어 **저온에서 24~48시간** 천천히 발효시키는 방식이다.

---

## 핵심 특징

| 구분 | 내용 |
|------|------|
| **발효 방식** | 액체 사전발효종 (Liquid Pre-ferment) |
| **수화율** | 100% (밀가루:물 = 1:1) |
| **발효 시간** | 24~48시간 저온 숙성 |
| **식감** | 겉은 바삭, 속은 쫄깃하고 촉촉함 |
| **풍미** | 견과류의 고소함 + 부드러운 크리미함 |
| **기공** | 크고 균일한 에어리(airy)한 구조 |

---

## 폴리쉬 도우의 장점

### 1. 식감
- **겉바속촉**: 크러스트는 얇고 바삭, 속은 쫄깃하고 촉촉
- **빵끝까지 맛있음**: 크러스트도 버리지 않고 먹게 됨
- 가볍고 공기감 있는 텍스처

### 2. 풍미
- 견과류의 **고소한 향**
- **크리미하고 부드러운** 맛 (요거트, 버터 뉘앙스)
- 밀가루 본연의 깊은 맛이 살아남

### 3. 소화
- 긴 발효 과정에서 **글루텐이 분해**되어 속이 편안함
- 발효 과정에서 생긴 **유산(Lactic Acid)**이 소화를 도움
- 먹고 나서도 더부룩하지 않음

### 4. 보존성
- 발효로 인한 산성도 상승 → 빵의 **신선도 오래 유지**

## 다른 도우 방식과 비교

| 구분 | 폴리쉬 (Poolish) | 비가 (Biga) | 사워도우 (Sourdough) |
|------|------------------|-------------|---------------------|
| **기원** | 프랑스 | 이탈리아 | 전통 발효빵 |
| **수화율** | 100% (묽음) | 45-60% (되직함) | 다양 |
| **발효시간** | 6~24시간 | 12~48시간 | 수일 (관리 필요) |
| **풍미** | 크리미, 고소함 | 깊고 새콤함 | 강한 신맛 |
| **식감** | 가볍고 부드러움 | 쫄깃하고 탄탄함 | 쫄깃, 밀도감 |
| **난이도** | 쉬움 | 보통 | 어려움 |

## 노아이디어 피자의 폴리쉬 도우

### 우리만의 방식
- **매일 매장에서 직접 반죽** (냉동 도우 ❌)
- **48시간 저온 숙성** (정성의 시간)
- 밀가루 + 물 + 효모가 **천천히 대화하며** 만들어낸 결과물

### 왜 폴리쉬인가?
> "프랜차이즈가 '냉동 도우의 대량생산'이라는 수동변속기를 밟고 있을 때, 우리는 매일 매장에서 반죽을 치대고 있었다."

---

## 콘텐츠별 활용 카피

### 🎯 한 줄 요약
> "48시간을 기다린 도우, 빵끝까지 남기지 않는 비밀"

### 📱 숏폼/릴스용

**후킹 카피:**
- "피자 크러스트 남기는 사람 손들어보세요 🙋‍♂️"
- "왜 우리 피자는 빵끝까지 다 먹냐고요?"
- "48시간 기다린 이유가 있습니다"

**본문 카피:**
- "폴리쉬 발효법. 밀가루와 물과 효모가 48시간 동안 천천히 대화합니다."
- "냉동 도우? 절대 NO. 매일 직접 반죽합니다."
- "겉바속촉은 기본, 먹고 나서도 속이 편해요."

### 📝 블로그/상세페이지용

**제목 안:**
- 노아이디어피자가 '폴리쉬 도우'에 집착하는 이유
- 48시간의 기다림, 프랜차이즈가 못 따라하는 도우의 비밀
- 빵끝까지 맛있는 피자의 과학: 폴리쉬 발효법

**본문 구성:**
1. 도우에 대한 고민 (왜 빵끝은 항상 남는가?)
2. 폴리쉬란 무엇인가 (프랑스 정통 발효법 설명)
3. 우리의 방식 (48시간, 매일 직접, 정성)
4. 결과 (식감, 풍미, 소화의 차이)

### 🖥️ 랜딩페이지용

**히어로 섹션:**
> **"48시간, 도우가 완성되는 시간"**
> 프랑스 정통 폴리쉬 발효법으로
> 겉은 바삭, 속은 쫄깃, 빵끝까지 맛있게.

**세부 섹션:**
- ⏰ **48시간 저온 숙성** — 밀가루와 효모의 느린 대화
- 🙅‍♂️ **냉동 도우 NO** — 매일 매장에서 직접 반죽
- 😌 **속 편한 피자** — 발효로 글루텐 분해, 더부룩함 없이

## 네이밍 아이디어 (도우 브랜딩)

### 컨셉: "기다림의 가치"를 담은 네이밍

| 네이밍 | 컨셉 설명 |
|--------|----------|
| **48도우** | 48시간 숙성을 강조, 숫자의 임팩트 |
| **슬로우 도우** | 느린 발효의 가치, 슬로우푸드 연상 |
| **라이브 도우** | 살아있는 발효, 매일 직접 반죽 강조 |
| **숙성 도우** | 정직하고 직관적인 표현 |
| **대화 도우** | "밀가루와 효모의 대화"라는 스토리텔링 |

### 컨셉: 지역/장인 느낌

| 네이밍 | 컨셉 설명 |
|--------|----------|
| **이수 명장 도우** | 이수 본점 + 장인 정신 |
| **이수 발효 도우** | 지역 + 기술력 조합 |
| **[지역명] 숙성 도우** | 확장 시 각 지역 네이밍 가능 |

### 컨셉: 감성/호기심 유발

| 네이밍 | 컨셉 설명 |
|--------|----------|
| **비밀 도우** | "이 맛의 비밀을 알 수 없게" 브랜드 철학 연결 |
| **미스터리 도우** | No Idea 컨셉과 연결 |
| **프렌치 크래프트 도우** | 프랑스 정통 기법 강조, 프리미엄 느낌 |

### 🏆 추천 조합

**1순위: "48도우"**  
- 기억하기 쉬움 (숫자)
- 소금우유도우처럼 직관적
- "왜 48이에요?" → 스토리텔링 연결

**2순위: "슬로우 도우" / "라이브 도우"**  
- 가치 전달 명확
- 트렌디한 슬로우푸드, 건강 연상

**3순위: "이수 명장 도우"**  
- 지역성 + 장인 정신
- 프랜차이즈 확장 시 "[지역] 명장 도우"로 변형 가능
