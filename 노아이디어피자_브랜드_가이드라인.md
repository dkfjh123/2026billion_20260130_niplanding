## 1. 브랜드 개요 (Brand Overview)

### 1.1 브랜드 기본 정보

| 항목 | 내용 |
|------|------|
| 브랜드명 | 노아이디어피자 (NO IDEA PIZZA) |
| 영문명 | No Idea Pizza |
| 슬로건 | "Our own pizza" |
| 업종 | 수제 피자 전문점 |
| 핵심 콘셉트 | 천연액종 폴리쉬 도우 + 독창적 토핑 조합 + 힙한 공간 경험 |

### 1.2 네이밍의 역설적 의미

**"No Idea"**라는 이름은 두 가지 의미를 담고 있습니다:

1. **결정 피로 해소**: 복잡한 고민 없이 편안하게 즐길 수 있는 피자
2. **놀라운 미식 경험**: 한 입 베어 물면 "어떻게 이런 맛이?"라며 그 비결을 알 수 없는 놀라움

### 1.3 브랜드 핵심 가치 (Core Values)

| 가치 | 설명 |
|------|------|
| **Freshness (신선함)** | 냉동 도우가 아닌 매일 직접 반죽하는 생도우와 신선한 원재료 |
| **Originality (독창성)** | 기존 프랜차이즈에서 시도하지 않은 과감한 토핑 조합 (용과, 꽈리고추 등) |
| **Experience (경험)** | 단순히 음식을 먹는 공간을 넘어 시각적 즐거움을 제공하는 '힙'한 공간 |

### 1.4 브랜드 철학

- 천연액종을 이용한 폴리쉬 도우로 **건강한 피자** 제공
- 신선한 재료와 **고품질 치즈** 사용
- 일반 프랜차이즈와 차별화된 **독창적인 토핑 조합**
- 합리적인 가격으로 **고퀄리티 피자** 제공

## 3. 기술적 차별점 (Technical Innovation)

### 3.1 폴리쉬(Poolish) 도우 공법

노아이디어피자 맛의 핵심은 **폴리쉬 발효법**에 기반한 도우입니다.

| 특징 | 상세 내용 | 기대 효과 |
|------|----------|----------|
| **사전 발효** | 밀가루와 물, 효모를 1:1 비율로 섞어 장시간 사전 발효 | 풍부한 유기산 생성으로 깊은 풍미 |
| **저온 숙성** | 급격한 발효 지양, 저온에서 천천히 숙성 | 글루텐 구조 안정화, 소화 용이 |
| **매장 직접 반죽** | 매일 매장에서 직접 반죽 | 신선하고 일정한 품질 |
| **쌀가루 첨가** | 도우에 쌀가루 추가 | 쫀득퐁신한 식감 |

**결과물**: 겉은 바삭, 속은 쫄깃한 '겉바속촉'의 정석

### 3.2 프리미엄 재료

| 재료 | 상세 |
|------|------|
| **치즈** | 99~100% 자연산 모짜렐라 기반, 라끌렛/체다/그라나 파다노 블렌딩 |
| **소스** | 갈릭 디핑, 렌치, 할라피뇨 마요 등 매장 자체 제조 |
| **피클** | 빙초산 미사용, 매일 직접 담근 수제 피클 |
| **야채** | 직접 손질한 신선한 재료 |
| **육류** | 두툼하고 품질 좋은 고기 |

| 항목 | 점수 |

### 8.2 긍정적 피드백

**맛**
- 토핑 퀄리티가 일반 프랜차이즈보다 훨씬 뛰어남
- 도우가 쫄깃하고 맛있음
- 치즈가 고소하고 양이 많음
- 끝까지 먹어도 질리지 않음
- 꽈리불고기 피자는 재방문 의사 100%

**서비스 & 공간**
- 주문 후 피자가 빨리 나옴
- 힙하고 트렌디한 인테리어
- 데이트하기 좋음
- SNS 인증샷 찍기 좋음
